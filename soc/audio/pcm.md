# PCM

## PCM编码：

>PCM 脉冲编码调制是`Pulse Code Modulation`的缩写。脉冲编码调制是数字通信的编码方式之一。主要过程是将话音、图像等模拟信号每隔一定时间进行取样，使其离散化，同时将抽样值按分层单位四舍五入取整量化，同时将抽样值按一组二进制码来表示抽样脉冲的幅值。

## 作用：

**主要是将`模拟信号`转化为`数字信号`**

## 实现：

抽样(Sampling)  --->  量化(quantizing)  --->   编码(Coding)
